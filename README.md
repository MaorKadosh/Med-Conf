# טיימר הכנס הלאומי לרפואה צבאית

אפליקציית ווב מותאמת לטאבלטים Samsung Galaxy Tab A8 (SM-X205) לניהול טיימר הדוברים בכנס הרב-אולמי של חיל הרפואה.

## שימוש

1. פתחו את הקובץ `index.html` בדפדפן כרום/Edge על גבי כל טאבלט.
2. במסך הניהול הוסיפו את המרצים לכל אולם באמצעות הטופס.
3. לחצו על "בחר טאבלט לאולם זה" כדי לשייך את הטאבלט לאולם הרצוי.
4. הפעילו את הטיימר, עברו בין מרצים והציגו את מסך הטיימר לקהל.
5. מומלץ ללחוץ על "מסך מלא" ו"מניעת שינה" בכל טאבלט.

המידע נשמר באופן מקומי (LocalStorage) בכל טאבלט כך שניתן לרענן את העמוד מבלי לאבד נתונים.

## פיצ׳רים עיקריים

- תמיכה ב־RTL ובשפה העברית.
- שני מסכים: ניהול וטיימר.
- ניהול רשימות מרצים לכל אחד מארבעת האולמות.
- טיימר עם פס התקדמות, לוגו הכנס וקרדיט למדור תקשוב משר״פ, 8280.
- תמיכה ב־Wake Lock (מניעת שינה) ובמסך מלא, עם הודעת גיבוי לדפדפנים שאינם תומכים.

## פיתוח

הפרויקט הינו סטטי לחלוטין. ניתן לפרוס על כל שרת קבצים סטטי או לפתוח ישירות מהקבצים המקומיים.

## אריזת APK מקומי לטאבלט

1. התקינו Android Studio 2023.1 ומעלה (או Gradle 8.3+ עם Android Gradle Plugin 8.3.2).
2. פתחו את התיקייה `android-app` מתוך Android Studio ובחרו **Import Project**.
3. ודאו שהפרויקט מסתנכרן בהצלחה (Android Studio ייצור באופן אוטומטי Gradle Wrapper אם חסר).
4. הקובץ `app/src/main/assets/` מכיל עותק עדכני של קבצי הווב. אם שיניתם את גרסת הווב, העתיקו מחדש את `index.html`, `styles.css`, `app.js` ואת תיקיית `assets/` אל מיקום זה לפני בנייה.
5. להרצת גרסת Debug על טאבלט מחובר, בחרו `Run > Run 'app'`.
6. ליצירת קובץ התקנה חתום ידנית (APK):
   - בחרו בתפריט `Build > Generate Signed Bundle / APK`.
   - צרו Keystore מקומי (פעם אחת) או בחרו קיים.
   - בחרו ב־`APK` ולאחר מכן ב־`Release`.
   - לחצו על `Finish` והמתינו ליצירת הקובץ `app-release.apk` תחת `android-app/app/build/outputs/apk/release/`.
7. העבירו את ה־APK שנוצר לטאבלט והתקינו ידנית (יש לאפשר התקנת אפליקציות ממקורות לא מוכרים בהגדרות הטאבלט).

### אפשרות בנייה באמצעות שורת פקודה

לאחר יצירת Gradle Wrapper (באמצעות Android Studio או `gradle wrapper` מקומי):

```bash
cd android-app
./gradlew assembleRelease
```

תמצאו את הקובץ המוכן בנתיב `app/build/outputs/apk/release/app-release.apk`.

ה־APK כולל WebView במצב מסך מלא עם מניעת שינה מובנית לטאבלט, כך שניתן להריץ את הטיימר גם ללא דפדפן.
